HTML and CSS

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

HTML code example:

<p>This is a paragraph element in HTML</p>

Cascading Style Sheets (CSS) is used to describe the presentation of a document written in HTML.

CSS code example:

p { text-align: center; }
JavaScript

JavaScript is one of the core technologies of the World Wide Web, alongside HTML and CSS.

JavaScript code example:

function greetMe(name) { alert("Hello " + name); } greetMe("World");
Python

Python is a high-level, general-purpose programming language known for its readability.

Python code example:

print("Hello, world!")
React

React is a free and open-source JavaScript library for building user interfaces.

React code example:

const root = ReactDOM.createRoot(document.getElementById("root")); root.render(<h1>Hello, world!</h1>);
APIs

An Application Programming Interface (API) allows different software applications to communicate with each other.

Common API examples:

  • Google Fonts
  • Font Awesome
  • Mapbox
  • Adobe
  • GitHub

(* Less-than and greater-than symbols are written as text so the code does not execute.)
