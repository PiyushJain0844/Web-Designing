Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language.

Inside a host environment, JavaScript can be connected to the objects of its environment.

console.log("Hello World");
What you should already know

  • A general understanding of the Internet
  • Good working knowledge of HTML
  • Some programming experience

JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different.

JavaScript does not have static typing.

var language = "JavaScript";
Hello world

Variables

Variables are symbolic names for values.

The names of variables are called identifiers.

  • var
  • let
  • const
let y = 5;
Declaring variables

You can declare variables in three ways.

var x = 42; let y = 13;
Variable scope

Scope determines the accessibility of variables.

JavaScript has global and local scope.

if (true) { var a = 5; }
Global variables

Global variables are properties of the global object.

window.value = 10;
Constants

Constants are read-only variables.

const PI = 3.14;
Data types

JavaScript defines seven data types.

  • Boolean
  • null
  • undefined
  • Number
  • String
  • Symbol
  • Object
typeof 42;
if...else statement

The if statement executes code if a condition is true.

The else statement executes code if the condition is false.

if (condition) { statement_1; } else { statement_2; }
while statement

A while loop executes as long as a condition is true.

The condition is evaluated before each iteration.

var n = 0; var x = 0; while (n < 3) { n++; x += n; }
  • First pass: n = 1, x = 1
  • Second pass: n = 2, x = 3
  • Third pass: n = 3, x = 6
Function declarations

  • Function name

  • Statements
